01. 데이터 이펙트 : 1부터 100까지 출력하기

for문을 이용해서 1-100까지 출력하는 예제입니다.

for(let i=1; i<=100; i++){ // let i =0 으로 초기화 한후 i 를 100 까지 1씩 증가
    console.log(i); // 값을 출력
    }

// 1,2,3,4,5,6~~~100
